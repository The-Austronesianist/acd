The Language section

The Language section contains all the words used in the ACD, including Roots, Loans and Noise, grouped alphabetically by language, as well as all the proto-forms used within sets, but not reconstructed in the ACD, grouped by their Proto-languages.

The language name used is generally the one from the major reference used for that language.

Alphabetization is based on the English alphabet, with non-English letters assigned to their closest English counterparts, so that, for example, 'Are'are is alphabetized as if it were spelled Areare.

Across the top of each page are the first-letter index links to the language name pages, like the one at the top of this page.

Key to parts of an entry

189. - Temporary language number. The number preceding each language name in this section is for the current version only, and it may change when a new language is included into the ACD. (The "current version" is the date in the upper right hand corner of the page, like "Updated: 5/24/2012".)

Gilbertese - Language Name. The language name is the name used for this language throughout the ACD. Language names in the ACD are occasionally modified, in which case the old name is listed as a cross-reference entry.

(167) - Word Count. The parenthetical number following the language name is the number of words from that language cited in ACD reconstructions (including Loans, Roots, Noise).

OC - Group Code. (Small-caps like "OC" for Oceanic), indicates the subgroup of the language and is a link to the list of all the languages in that group.

(Sabatier 1971) - Reference citation is for the major source(s) of the data for that language, and is a link to the full reference on the References page.

[gil] - ISO code. The bracketed lower-case 3-letter code is the the code assigned to the language by the ISO 639-3 standard (ISO 2007), and a link to the corresponding page at Ethnologue, Languages of the World.

(Kiribati) - Ethnologue Name. The parenthesized language name, like "(Kiribati)", is the name used at Ethnologue, when it differs from the ACD.

Kiribati - Geo-Political Location. The country or area name is the geo-political designation of the language, following Ethnologue.

(ma)-ma-iu - Word, a link to the subset of a Cognate Set in which the word used in a reconstruction.

living, flourishing, fresh, prosperous - the Gloss.

(PMP: *ma-qudip) - Subset, a link to the subset in which the word appears.

*qudip - Cognate Set, which is a link to the cognate set.

Gilbertese (20 / 167) - The Progress Line is to assist in navigating long entries, and shows the current line number over the total word count for that language.
